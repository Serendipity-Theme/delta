# Serendipity for delta

Elegant, minimal, and clean color palette for your tools.

See other interfaces at the [official website](https://www.michaelandreuzza.com/vscode/serendipity/).

## Available themes

- **Midnight** — dark
- **Morning** — light
- **Sunset** — dark

## Installation

Include a fragment in `~/.gitconfig` under `[include]` or merge the `[delta]` section.

Available files: `serendipity-midnight.gitconfig`, `serendipity-morning.gitconfig`, `serendipity-sunset.gitconfig`.
